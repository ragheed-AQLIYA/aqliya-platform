# Strategic Audit Skill

## Audit Standard

Analyze like a senior enterprise platform reviewer.

No compliments.
No demo bias.
No optimistic assumptions.

Cover:
- strategy
- product architecture
- technical architecture
- data/governance
- GTM
- operational readiness
- enterprise risk
- product boundaries
- proof gaps
- next execution sequence

## Sources

- `docs/official/` v1.1
- `docs/source-of-truth/PRODUCT_STATUS_MATRIX.md`
- `docs/source-of-truth/ROUTE_STRATEGY.md`
- current code behavior (verify, do not assume)

## Output

- harsh reality
- strengths
- weaknesses
- P0/P1/P2 risks
- recommended sequence (technical detail, not calendar estimates)

Do not edit files unless explicitly asked.
