# AQLIYA v0.1 Demo Safety Guide

## Safe to Demo

| Area                   | How to Present                                                              | What Not to Claim                                                      |
| ---------------------- | --------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| AuditOS                | Present as the first proof product and pilot-ready governed audit workspace | Do not claim production-hardened enterprise rollout                    |
| Public marketing site  | Present as company and product positioning                                  | Do not imply every marketed product is implemented                     |
| Custom product inquiry | Present as live commercial intake flow                                      | Do not imply it provisions an implemented custom product automatically |

## Demo with Explanation

| Area                | Required Explanation                                                        | Risk                            |
| ------------------- | --------------------------------------------------------------------------- | ------------------------------- |
| DecisionOS          | Real active adjacent system; some areas are stronger than others            | Overstating production maturity |
| Office AI Assistant | Governed shared application, deterministic today, not a generic chatbot     | Overclaiming AI/runtime breadth |
| Sunbul              | Real custom/client-specific workspace, not a default platform product claim | Product-boundary confusion      |

## Safe to Demo with Explanation (LocalContentOS)

| Area           | Required Explanation                                                                  | Risk                                     |
| -------------- | ------------------------------------------------------------------------------------- | ---------------------------------------- |
| LocalContentOS | L5 pilot-ready with conditions; real workspace and mutations; not L6; text/CSV export | Overclaiming production or certification |

## Internal Only

| Area          | Reason                                                             |
| ------------- | ------------------------------------------------------------------ |
| workflowos    | Alias/duplicate custom workflow surface over Sunbul implementation |
| organizations | Protected prototype/internal preview only                          |
| settings      | Generic local-state preview only                                   |
| monitoring    | Internal diagnostics/operator surface                              |

## Do Not Present as Implemented

| Area               | Correct Framing                                   | Forbidden Framing                    |
| ------------------ | ------------------------------------------------- | ------------------------------------ |
| auditos demo route | Public guided demo with mock data                 | Live AuditOS workspace               |
| SalesOS            | Prototype / internal preview                      | Implemented SalesOS product          |
| LocalContentOS L6  | L5 pilot-ready with conditions; text/CSV export   | Production-hardened; binary PDF/XLSX |
| SimulationOS       | Marketing/category label or DecisionOS capability | Standalone SimulationOS product      |
| On-Prem            | Strategic deployment direction                    | Available production On-Prem package |
| Air-Gapped         | Strategic deployment direction                    | Available Air-Gapped deployment      |
| Local AI           | Strategic runtime direction                       | Live local AI runtime                |
| AQLIYA Studio      | Strategic custom systems layer                    | Implemented Studio builder           |
