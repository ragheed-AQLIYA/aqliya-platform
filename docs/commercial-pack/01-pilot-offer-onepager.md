# 🟢 AuditOS — عرض البايلوت التجريبي

## النظام باختصار

منصة رقمية لتنظيم أعمال المراجعة والتدقيق المالي. يبدأ النظام من استلام ميزان المراجعة (Trial Balance)، مرورًا بالتصنيف (Mapping)، وإعداد القوائم المالية والإيضاحات، وانتهاءً باعتماد الشريك ومراقب الجودة، مع ربط الأدلة والملاحظات والتوصيات في مسار واحد واضح.

المبدأ: **الذكاء الاصطناعي يساعد. الإنسان يقرر. الدليل يحكم.**

---

## 💼 من يستفيد؟

- مكاتب المحاسبة والمراجعة (خاصة IFRS for SMEs)
- فرق المراجعة المتوسطة والصغيرة
- الشريك المراجع ومدير التدقيق ومراقب الجودة

---

## ⚡ أبرز القدرات الجاهزة للتجربة

| المجال | القدرة |
|--------|--------|
| ميزان المراجعة | رفع CSV/XLSX وعرض الحسابات والانحرافات تلقائيًا |
| التصنيف المحاسبي | ربط كل حساب بدليل حسابات مرجعي (قابل للتعديل) |
| القوائم المالية | إنشاء قائمة الدخل والمركز المالي والتغير في حقوق الملكية |
| الإيضاحات | إدارة الإيضاحات مع إظهار النواقص واقتراحات مسودة |
| الأدلة | دورة حياة كاملة (مطلوب ← مرفوع ← مقبول ← مرفوض) |
| الملاحظات | تسجيل ملاحظات بأربعة مستويات خطورة مع ربطها بالأدلة |
| التوصيات | توصيات مرتبطة بالملاحظات مع مستوى المخاطرة |
| المراجعة | تعليقات موجهة لكل عنصر (قائمة / إيضاح / ملاحظة) |
| بوابة الاعتماد | ٥ شروط تمنع الاعتماد قبل الاكتمال |
| المراجعة التحليلية | كشف آلي للانحرافات والأرصدة غير المعتادة |
| الإسناد الزمني | سجل كامل بكل إجراء (18+ نوع حدث) |
| قابلية التتبع | عرض مسار أي رقم من المصدر إلى القائمة النهائية وبالعكس |

---

## 🔍 لماذا هذا العرض؟

هذه فرصة **بايلوت محدود** (Pilot) لاختبار النظام في بيئة عمل حقيقية على بيانات فعلية، بمشاركة فريق المراجعة لديكم، مع دعم كامل من فريق AQLIYA طوال فترة التجربة.

الهدف: التحقق من جودة سير العمل، وتنظيم الأدلة، وربط الملاحظات بالقوائم، قبل الإطلاق الإنتاجي الكامل.

## 📋 شروط العرض

| البند | التفصيل |
|-------|---------|
| النطاق | ارتباط واحد (Engagement) لمنشأة واحدة |
| البيانات | ميزان مراجعة من ٢٠–١٠٠ حساب |
| المدة | أسبوعان إلى ٤ أسابيع (حسب اختياركم) |
| التكلفة | **بدون مقابل مالي** خلال فترة البايلوت |
| الدعم | تدريب وتوجيه وجلسات أسبوعية مع فريق AQLIYA |
| المخرجات | تقرير تقييم نهائي وخطة تحول للإنتاج |

---

## ⚠️ المهم أن تعرفه

1. النظام **لا يستبدل المراجع** — هو أداة تنظيم وتسريع وضبط جودة.
2. كل اقتراحات الذكاء الاصطناعي هي **مسودات** تحتاج مراجعة واعتماد بشري.
3. التخزين حاليًا **محلي** في بيئة البايلوت (وليس سحابيًا).
4. الإخراج بصيغة JSON/XLSX. دعم PDF للعربية محدود حاليًا.
5. لا يوجد بوابة عميل خارجية بعد. رفع الملفات يتم من طرفكم داخل المنصة.
6. هذا بايلوت تقييمي — وليس اشتراكًا تجاريًا نهائيًا.
