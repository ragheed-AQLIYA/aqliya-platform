# Longitudinal Governance Observation

Track governance behavior across 5 simulated pilot sessions. Each session represents one engagement review.

## Session 1: First Exposure
Reviewer notices banner, opens governance panel, reads provenance. U-shape: entry glance → deep work → exit confirmation.

## Session 2: Familiar
Reviewer glances at banner, does NOT open panel. Goes directly to work. Governance is background context.

## Session 3: Routine
Banner barely noticed. Governance tooltip hovered briefly. Panel not opened. Reviewer trusts the system.

## Session 4: Ambiguity Event
Evidence conflict. Reviewer opens governance panel to check doctrine. Escalation triggered. Governance helped resolve uncertainty.

## Session 5: Stable Routine
Banner peripheral. Panel not needed. Reviewer works efficiently with implicit governance trust.

## Key Finding
Governance usage decreases over time but trust remains. Governance reappears during uncertainty — this is the healthy pattern.
