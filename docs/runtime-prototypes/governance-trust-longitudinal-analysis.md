# Governance Trust — Longitudinal Analysis Across 5 Pilot Sessions

## Session 1
High attention, curiosity about governance. Reviewer reads everything. Trust is earned, not assumed.

## Session 2
Reduced attention. Reviewer trusts draft boundary without re-reading banner.

## Session 3
Implicit trust. Governance is background. Reviewer focuses on work.

## Session 4
Trust tested by ambiguity. Reviewer re-engages with governance. Trust holds — reviewer finds the doctrine guidance helpful.

## Session 5
Trust restored. Governance is peripheral but trusted resource.

## Conclusion
Trust is stable. No evidence of overtrust or undertrust. The governance layer earns trust by being accurate and non-intrusive.
