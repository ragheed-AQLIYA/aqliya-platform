# Evidence Ambiguity Patterns

Evidence ambiguity occurred in 2 of 5 sessions:

- **Session 2**: Partial evidence for a material account. Reviewer flagged it, requested more evidence.
- **Session 4**: Conflicting evidence. Escalation triggered. Reviewer investigated and resolved.

The existing evidence state badges (missing/uploaded/reviewed/accepted) handled these cases without requiring additional governance UI. The governance panel's provenance summary ("Evidence: partial") provided context but was not the primary signal — the existing evidence page did that.
