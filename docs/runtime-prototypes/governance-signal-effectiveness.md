# Governance Signal Effectiveness Assessment

## 1. DraftOnlyBanner (amber card "Draft Only — Human review required")

| Criterion | Assessment |
|-----------|------------|
| Noticed? | Yes. High-contrast amber background draws attention immediately on page load. |
| Understood? | Yes. Clear language — reviewer understood the statement is not final. |
| Trusted? | Yes. Consistent positioning reinforces reliability. |
| Useful? | Yes. Prevents premature sign-off by framing the page state upfront. |
| Ignored? | No visible ignoring observed. The banner is above the fold and cannot be dismissed. |
| Distracting? | No. It is static and does not animate or flash. |

## 2. Governance Context shield button with tooltip ("2 doctrine refs, 2 governance rules, human review required")

| Criterion | Assessment |
|-----------|------------|
| Noticed? | Yes. The shield icon is distinctive and positioned near the statement header. |
| Understood? | Partially. The tooltip text is clear but the icon alone may not convey "governance context" to new users. Tooltip dependency means hover is required for understanding. |
| Trusted? | Yes. The count display (2 refs, 2 rules) gives concrete, auditable information. |
| Useful? | Yes. Provides a quick summary of governance weight without opening the panel. |
| Ignored? | No. Reviewer hovered over it to read the tooltip during the session. |
| Distracting? | No. Compact, non-intrusive. |
