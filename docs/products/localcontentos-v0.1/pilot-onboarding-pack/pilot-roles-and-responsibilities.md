# Pilot Roles & Responsibilities — LocalContentOS v0.1

## AQLIYA Team

| Role                         | Responsibility                                                                                |
| ---------------------------- | --------------------------------------------------------------------------------------------- |
| **AQLIYA Pilot Lead**        | Overall pilot coordination, customer communication, timeline management, escalation point     |
| **AQLIYA Technical Lead**    | Environment setup, user account creation, data loading, CSV import, technical troubleshooting |
| **AQLIYA Product Analyst**   | Data validation, classification guidance, findings review support, report preparation         |
| **AQLIYA Executive Sponsor** | High-level customer relationship, conversion discussion, commercial alignment                 |

## Customer Team

| Role                                 | Responsibility                                                                                                 |
| ------------------------------------ | -------------------------------------------------------------------------------------------------------------- |
| **Customer Sponsor**                 | Executive ownership of pilot, resource allocation, removes internal blockers, participates in decision meeting |
| **Procurement / Finance Owner**      | Provides spend records, supplier master data, validates spend categories and amounts                           |
| **Local Content / Compliance Owner** | Provides classification rules, validates supplier locality, reviews findings, owns the assessment              |
| **Reviewer**                         | Reviews the completed assessment, provides comments, returns for revision if needed                            |
| **Approver**                         | Makes final approval or rejection decision, records decision rationale                                         |
| **Technical Contact**                | Liaises with AQLIYA technical lead, facilitates data extraction, manages user accounts                         |

| Activity                 | AQLIYA Lead | AQLIYA Tech | Customer Sponsor | Procurement | LC Owner | Reviewer | Approver | Tech Contact |
| ------------------------ | ----------- | ----------- | ---------------- | ----------- | -------- | -------- | -------- | ------------ |
| Kickoff meeting          | A           | C           | A                | C           | C        | I        | I        | C            |
| Data extraction          | I           | C           | A                | R           | C        | I        | I        | R            |
| Data loading             | I           | R           | I                | I           | I        | I        | I        | C            |
| Supplier register review | C           | C           | I                | C           | A        | I        | I        | I            |
| Spend classification     | C           | C           | I                | C           | R        | I        | I        | I            |
| Evidence review          | C           | C           | I                | C           | R        | I        | I        | I            |
| Findings drafting        | R           | I           | I                | C           | A        | I        | I        | I            |
| Review submission        | C           | C           | I                | I           | C        | R        | I        | I            |
| Approval decision        | I           | I           | C                | I           | C        | I        | R        | I            |
| Report generation        | C           | R           | I                | I           | A        | I        | I        | I            |
| Closeout meeting         | R           | C           | A                | I           | C        | I        | I        | I            |

**R** = Responsible (does the work)  
**A** = Accountable (approves the work)  
**C** = Consulted (provides input)  
**I** = Informed (receives updates)
