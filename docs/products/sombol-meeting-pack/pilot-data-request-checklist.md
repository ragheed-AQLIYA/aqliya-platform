# Sombol — Pilot Data Request Checklist

**المرحلة:** Cloud Pilot
**نوع البيانات:** بيانات تجريبية وآمنة فقط — لا بيانات حقيقية للعملاء في المرحلة التجريبية إلا بعد الاتفاق.

---

## Safe Demo Data Request

| # | نوع البيانات | الهدف | المطلوب |
|---|---|---|---|
| 1 | **Trial Balance (ميزان مراجعة)** | اختبار AuditOS | ملف Excel بميزان مراجعة تجريبي (أرقام وهمية) |
| 2 | **Excel sample** | اختبار Office AI extraction | ملف Excel ببيانات وهمية (مبيعات، مصروفات، إلخ) |
| 3 | **PDF sample** | اختبار PDF extraction | ملف PDF بنص عربي (غير ممسوح ضوئياً) |
| 4 | **Word sample** | اختبار DOCX extraction | ملف Word بتنسيق بسيط |
| 5 | **CSV sample** | اختبار CSV extraction | ملف CSV ببيانات وهمية |
| 6 | **Client workspace names** | إعداد بيئة العرض | أسماء وهمية للعملاء (مثلاً: "شركة الخليج"، "مؤسسة نجد") |
| 7 | **User list for pilot** | إنشاء حسابات المستخدمين | الأسماء + البريد الإلكتروني + الدور (مدير/مدقق/مشاهد) |

## Data Anonymization Guidance

إذا اخترتم استخدام بيانات شبه حقيقية للاختبار، يرجى تطبيق التالي:

| الإجراء | التفاصيل |
|---|---|
| تغيير أسماء العملاء | استخدم أسماء وهمية |
| تغيير أسماء الموظفين | استخدم أسماء وهمية |
| تعديل الأرقام المالية | غيّر القيم — لا تستخدم أرقاماً حقيقية |
| إزالة المعرفات الشخصية (PII) | لا أرقام هوية، لا جوازات سفر، لا عناوين |
| إزالة البيانات المصرفية | لا أرقام حساب، لا IBAN |
| إزالة البيانات الحكومية | لا سجلات حكومية أو تنظيمية |

---

## What NOT to Send in Pilot

| ❌ لا ترسل | السبب |
|---|---|
| بيانات عملاء حقيقية بدون موافقة خطية | مسؤولية قانونية |
| مستندات ممسوحة ضوئياً (صور) | OCR غير مطبّق — لن يتم استخراج النص |
| بيانات من جهات رقابية (هيئة السوق المالية، إلخ) | غير مسموح بها في Pilot |
| ملفات أكبر من 10 ميجابايت | حد المنصة الحالي |
| ملفات تنفيذية (.exe, .msi, إلخ) | ممنوعة لأسباب أمنية |
| بيانات شخصية (PII) | غير مسموح بها في Pilot |
| بيانات مصرفية حقيقية | غير مسموح بها |

---

## Recommended Approach

| الخطوة | الإجراء |
|---|---|
| 1 | Sombol يرسل قائمة بأسماء العملاء الوهمية |
| 2 | AQLIYA يقوم بإنشاء ClientWorkspaces |
| 3 | Sombol يرسل ملفات تجريبية (Excel, PDF, Word) |
| 4 | AQLIYA يقوم بتحميل الملفات واختبار الاستخراج |
| 5 | Sombol يرسل قائمة المستخدمين التجريبيين |
| 6 | AQLIYA يقوم بإنشاء الحسابات وإرسال الدعوات |

> **مدة التجهيز:** 3–5 أيام عمل من استلام البيانات.
