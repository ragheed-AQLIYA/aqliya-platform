# AuditOS CRM Qualification Fields

Source of truth:

1. `docs/products/auditos-product-packaging.md`
2. `docs/products/auditos-outbound-kit/`

## Core Fields

1. Account name
2. Contact name
3. Contact role
4. Segment
5. Geography
6. Internal owner

## Qualification Fields

1. Main pain
2. Current workflow summary
3. Current tools
4. Primary use case
5. Buying trigger
6. Urgency level
7. Trust concern
8. Decision-maker identified? (Yes/No)
9. Sponsor identified? (Yes/No)
10. Demo fit? (High/Medium/Low)
11. Pilot fit? (High/Medium/Low)

## Commercial Fields

1. Last contact date
2. Next step
3. Next step date
4. Lead status
5. Deal stage
6. Risk note
7. Objection summary

## Usage Rule

لا تعتبر الفرصة مؤهلة فعليًا إذا كانت الحقول التالية ناقصة:

1. segment
2. main pain
3. current workflow summary
4. buyer or sponsor
5. next step
