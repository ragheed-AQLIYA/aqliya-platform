## Slide 2: Current Pain

| Field                 | Content                                                                                                                                                                                                      |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Title**             | How Organizations Track Local Content Today                                                                                                                                                                  |

| Field                 | Content                                                                                                                                                                                                                                                                                           |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Title**             | The Cost of Manual Local Content Tracking                                                                                                                                                                                                                                                         |

The answer is usually uncomfortable.                                                                                                                                                         |

## Slide 6: Evidence and Review Model

| Field                 | Content                                                                                                                                                                                                               |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Title**             | Evidence: The Core of Trust                                                                                                                                                                                           |
| **Key Message**       | A local content claim without evidence is just an opinion. Every classification must be backed by documentation.                                                                                                      |
| **Bullets**           | • High confidence: Official certificates, valid CR<br>• Medium confidence: Supplier declarations, invoices<br>• Low confidence: Self-declaration without proof<br>• None: Best guess — excluded from verified metrics |
| **Visual Suggestion** | Confidence level pyramid or gauge visualization                                                                                                                                                                       |
| **Speaker Notes**     | Ask: "What percentage of your current local content claims would be High confidence?" Most organizations realize they don't know. This is the value of evidence governance.                                           |
