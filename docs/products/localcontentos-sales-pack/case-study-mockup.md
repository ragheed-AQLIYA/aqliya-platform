## Customer Profile

| Field                        | Value                                                                          |
| ---------------------------- | ------------------------------------------------------------------------------ |
| **Company**                  | National Industrial Services Co. (NISCO) — fictional                           |
| **Industry**                 | Industrial services, equipment maintenance, and spare parts supply             |
| **Size**                     | ~2,000 employees, ~500M SAR annual procurement                                 |
| **Local Content Obligation** | Required by multiple government and semi-government contracts                  |
| **Previous Process**         | Excel-based tracking across 3 separate spreadsheets                            |
| **Pain Point**               | Could not produce evidence-backed local content numbers for tender submissions |

---

## Initial Problem

NISCO had three separate Excel files:

1. **Vendor list** — 120 suppliers, manually classified as local/non-local, no CR numbers tracked
2. **Procurement log** — 850 transactions per year, classified by category, no vendor linkage
3. **Contract register** — 45 active contracts, LC commitments documented in free-text notes

The local content officer spent **2 weeks every quarter** compiling these into a report. No one could audit the numbers. Evidence was scattered across email attachments and a shared drive.

## Evidence Gaps Found

| Gap                    | Count           | Impact                                   |
| ---------------------- | --------------- | ---------------------------------------- |
| Missing CR numbers     | 42 vendors      | 18 vendors classified as Local had no CR |
| Expired certificates   | 6 vendors       | LC certificates expired for 3–11 months  |
| Self-declaration only  | 15 vendors      | Low confidence — no supporting docs      |
| Invoices missing       | 23 transactions | Evidence not linked to spend             |
| LC clause undocumented | 5 contracts     | Verbal commitment with no written clause |

---

## Classification Issues Found

| Issue                        | Count           | Detail                                                            |
| ---------------------------- | --------------- | ----------------------------------------------------------------- |
| Vendors misclassified        | 8               | Classified as Local but no Saudi CR or operations                 |
| Mixed classification unclear | 5               | Local distributors for international brands — attribution unclear |
| Undetermined vendors         | 12              | No classification basis documented                                |
| Spend unclassified           | 18 transactions | Vendor not in master list                                         |

---

## Management Decisions Enabled

| Decision                      | Outcome                                                                 |
| ----------------------------- | ----------------------------------------------------------------------- |
| Reclassification of 8 vendors | Reclassified from Local to Non-Local or Mixed — 1.8M SAR spend affected |
| Evidence collection campaign  | Procurement team assigned to collect CRs from 42 vendors                |
| Certificate renewal plan      | 6 expired certificates flagged for renewal                              |
| LC clause formalization       | Legal team instructed to add written LC clauses to 5 contracts          |
| Classification policy drafted | Formal classification criteria documented for the first time            |

## Pilot Outcome

| Metric                    | Before Pilot         | After Pilot                      |
| ------------------------- | -------------------- | -------------------------------- |
| Local content % (raw)     | ~72%                 | 58% (verified)                   |
| High-confidence vendors   | 0%                   | 45%                              |
| Medium-confidence vendors | 0%                   | 30%                              |
| Low-confidence vendors    | 100% (self-declared) | 15%                              |
| Evidence gaps tracked     | None — no system     | 42 documented with owners        |
| Audit trail               | None                 | Full classification decision log |

**The real number was 14 percentage points lower than the Excel number** — but it was evidence-backed and defensible.

## Reusable Proof Points

- **Evidence gap** — 42 of 120 suppliers (35%) had no CR on file — a common problem across organizations
- **Classification correction** — 8 of 120 (6.7%) were misclassified — significant impact on the reported number
- **Confidence gap** — 100% of vendors were self-declared with no evidence before the pilot — only 15% remained Low confidence after
- **Management decisions enabled** — 4 concrete decisions made based on pilot findings
- **Pilot success score** — 2.75/3.0 — demonstrated that a governed approach produces actionable results
