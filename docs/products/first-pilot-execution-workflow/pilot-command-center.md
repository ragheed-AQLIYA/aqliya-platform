# Pilot Command Center — AQLIYA

## Purpose

Single-page dashboard for every active pilot. Provides instant status visibility for the AQLIYA team.

## When to Use

- Created at pilot kickoff
- Updated after every customer session or internal decision
- Reviewed at weekly operating rhythm

## Inputs

- Pilot scope template (`pilot-control-pack/pilot-scope-template.md`)
- Customer data handling rules (`pilot-control-pack/customer-data-handling-rules.md`)
