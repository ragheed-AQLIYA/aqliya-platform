## Purpose

هذا المستند يجمع أنواع proof القابلة لإعادة الاستخدام عبر العملاء والـ prospects.

## Proof Asset Types

1. time saved
2. workflow clarity
3. review improvement
4. traceability improvement
5. stakeholder confidence
6. reduced rework

## Proof Capture Format

لكل proof سجّل:

1. account
2. segment
3. proof type
4. exact statement or metric
5. source
6. whether it is public / anonymized / internal only

## Reuse Categories

1. sales deck proof
2. outbound proof
3. proposal proof
4. reference call prep
5. case study drafting

## Rule

الـ proof library يجب أن تكون قابلة للبحث حسب:

1. segment
2. use case
3. proof type
4. permission level
