# AuditOS Pilot Extension Decision Framework

Source of truth:

1. `docs/products/auditos-live-pilot-management/pilot-success-review.md`
2. `docs/products/auditos-first-customer-loop/pilot-to-paid-conversion-memo.md`

## Purpose

هذا المستند يحدد متى نمدد pilot ومتى لا نفعل ذلك.

## Valid Extension Reasons

1. value ظهرت لكن scope الأصلية كانت ضيقة
2. stakeholder engagement تأخرت لكن ما زال fit جيدًا
3. نحتاج إثبات use case ثانية قريبة من الأولى

## Weak Extension Reasons

1. لأن القرار صعب فقط
2. لأن buyer غير واضحة
3. لأن القيمة غير موجودة ونحاول فقط كسب وقت

## Extension Paths

### 1. Short Extension

1. مدة قصيرة
2. هدف محدد جدًا
3. success criterion واحدة أو اثنتان

### 2. Scope Refinement Extension

1. إعادة ضبط use case
2. تضييق workflow
3. اختبار الجزء الذي ظهرت فيه value أقوى

## No-Extension Criteria

لا تمدد إذا:

1. no-fit واضح
2. sponsor ضعيفة
3. no evidence of value
4. العميل لا يتحرك فعليًا

## Extension Decision Rule

التمديد يجب أن يجيب على سؤال واحد:

"ما الذي سنثبته في التمديد ولم يُثبت بعد؟"

إذا لم توجد إجابة واضحة، فالتمديد غير مبرر.
