# LocalContentOS — ICP and Use Cases

**Status:** Discovery / Planned (not implemented)
**Version:** 1.0 — Discovery Pack

---

## ICP Segments

### 1. Large Saudi Private Companies

| Dimension              | Detail                                                                                                                                                                        |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Pain**               | No governed system to measure and report local content. Manual Excel-based tracking across multiple business units. Inconsistent classification of local vs. non-local spend. |
| **Trigger**            | Annual reporting requirement, tender requirement from government client, internal compliance mandate.                                                                         |
| **Buyer**              | CFO, Procurement Director, Local Content Manager                                                                                                                              |
| **User**               | Procurement team, supply chain analysts, local content officers                                                                                                               |
| **Required Data**      | Vendor master, purchase orders, invoices, contracts, supplier classification                                                                                                  |
| **Strongest Use Case** | Periodic local content reporting for internal management and tender submissions                                                                                               |
| **Pilot Angle**        | One reporting period, one procurement category, sample suppliers                                                                                                              |
| **Risk / Objection**   | "We already track this in Excel / ERP." — Show evidence governance gap, audit trail value, workflow efficiency.                                                               |

---

### 2. Government Suppliers

| Dimension              | Detail                                                                                                                                                                            |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Pain**               | Need to demonstrate local content contribution to maintain government contracts. Scattered evidence across departments. Manual compilation before each tender or reporting cycle. |
| **Trigger**            | Tender submission requiring local content declaration. Government audit or review of local content commitments.                                                                   |
| **Buyer**              | Head of Government Affairs, Contracts Director, Local Content Officer                                                                                                             |
| **User**               | Bidding team, procurement team, compliance team                                                                                                                                   |
| **Required Data**      | Contract register, supplier certificates, spend data by contract, workforce data                                                                                                  |
| **Strongest Use Case** | Preparing evidence-backed local content declarations for government tenders                                                                                                       |
| **Pilot Angle**        | One major contract, track and report its local content from procurement to delivery                                                                                               |
| **Risk / Objection**   | "Government has its own portal / system." — Position as internal governance and preparation system, not a submission portal.                                                      |

---

### 3. Semi-Government Contractors

| Dimension              | Detail                                                                                                                                            |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Pain**               | Pressure from semi-government clients to show local content. Multiple clients with different reporting formats. No standardized internal process. |
| **Trigger**            | Contractual local content commitment, quarterly reporting obligation to client.                                                                   |
| **Buyer**              | CEO, VP of Operations, Contracts Manager                                                                                                          |
| **User**               | Procurement analysts, contract administrators, project managers                                                                                   |
| **Required Data**      | Supplier contracts, sub-contractor agreements, procurement logs, material sourcing data                                                           |
| **Strongest Use Case** | Multi-client local content reporting from a single governed system                                                                                |
| **Pilot Angle**        | One quarter, one client contract, demonstrate governance and efficiency gains                                                                     |
| **Risk / Objection**   | "Our clients accept our current reports." — Raise the bar to evidence-backed, audit-ready reports rather than self-declarations.                  |

---

### 4. Procurement-Heavy Companies

| Dimension              | Detail                                                                                                                                             |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Pain**               | High procurement volume makes manual local content tracking impractical. Need automated classification and evidence management at scale.           |
| **Trigger**            | Internal audit finding on local content tracking gaps. New regulation or policy requiring local content monitoring.                                |
| **Buyer**              | Chief Supply Chain Officer, VP Procurement                                                                                                         |
| **User**               | Procurement managers, category managers, supplier relationship managers                                                                            |
| **Required Data**      | Full procurement ledger, supplier master, contracts, purchase orders, goods receipts                                                               |
| **Strongest Use Case** | Automated spend classification and locality tagging across thousands of transactions                                                               |
| **Pilot Angle**        | One procurement category (e.g., IT hardware, construction materials) with high transaction volume                                                  |
| **Risk / Objection**   | "Our ERP can do this with customization." — ERP lacks governed evidence linking, review workflow, and audit trail purpose-built for local content. |

---

### 5. Companies with Local Content Obligations

| Dimension              | Detail                                                                                                                                    |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Pain**               | Formal local content obligations from regulators, licensors, or strategic partners. Need auditable trail of local content claims.         |
| **Trigger**            | Regulatory filing deadline, partner audit, license renewal requiring local content proof.                                                 |
| **Buyer**              | Compliance Director, Legal Counsel, Head of Regulatory Affairs                                                                            |
| **User**               | Compliance analysts, local content team, internal audit                                                                                   |
| **Required Data**      | Regulatory filings, commitment letters, supplier contracts, local content certificates, workforce records                                 |
| **Strongest Use Case** | Building an auditable evidence package for regulatory local content filings                                                               |
| **Pilot Angle**        | One filing period, build complete evidence trail, demonstrate audit-readiness                                                             |
| **Risk / Objection**   | "We submit directly to the regulator." — Internal governance is separate from regulatory submission; the system prepares what you submit. |

---

### 6. CFO / Procurement / Compliance Teams (Decision-Maker Level)

| Dimension              | Detail                                                                                                                                                                                         |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Pain**               | No visibility into local content performance across the organization. Cannot answer basic questions: What is our local content percentage? Is it improving? Are our claims backed by evidence? |
| **Trigger**            | Board request for local content metrics. External auditor question.                                                                                                                            |
| **Buyer**              | CFO, Chief Compliance Officer                                                                                                                                                                  |
| **User**               | Finance team, procurement analysts, compliance analysts                                                                                                                                        |
| **Required Data**      | Aggregated spend by classification, supplier locality summary, exception reports, trend data                                                                                                   |
| **Strongest Use Case** | Executive dashboard showing local content KPIs, trends, and risk areas                                                                                                                         |
| **Pilot Angle**        | Summary-level pilot with executive dashboard and gap analysis report (no detailed transaction processing needed for initial buy-in)                                                            |
| **Risk / Objection**   | "Give me a dashboard, not a system." — The dashboard is the output; the system ensures the data behind it is governed and auditable.                                                           |
