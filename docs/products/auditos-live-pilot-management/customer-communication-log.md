# AuditOS Customer Communication Log

Source of truth:

1. `docs/products/auditos-outbound-kit/`
2. `docs/products/auditos-sales-ops/`

## Purpose

هذا المستند يسجل كل تواصل مهم مع عميل pilot بشكل مبسط وواضح.

## Suggested Fields

| Field | Description |
|---|---|
| Date | تاريخ التواصل |
| Account | اسم العميل |
| Contact | اسم الشخص |
| Role | دوره |
| Channel | email / WhatsApp / call / meeting |
| Purpose | why this touch happened |
| Summary | ملخص ما قيل |
| Concern / Risk | أي قلق أو اعتراض ظهر |
| Commitment Made | أي التزام تم أخذه أو إعطاؤه |
| Next Step | الخطوة التالية |
| Owner | المسؤول الداخلي |

## Logging Rule

سجّل كل تواصل إذا كان فيه واحد أو أكثر من التالي:

1. decision point
2. objection
3. scope change
4. pilot commitment
5. risk escalation
6. next-step agreement

## Communication Discipline

1. لا تترك اجتماعًا بدون log مختصر
2. لا تعتمد على الذاكرة لتحديد ما تم الاتفاق عليه
3. كل communication log يجب أن تنتهي بـ next step واضحة
