# AQLIYA Commercial Documentation

## Current Role

This folder contains commercial and go-to-market materials for AuditOS. Some files overlap with `docs/commercial-pack/` and `docs/product/auditos-*` directories. Do not treat all files as equally current — see the overlap warnings below.

## Recommended Reading Order

### For AuditOS pilot preparation

- `demo-storyline/` — Modular demo storyline (opening pitch, pain statement, demo flow, screen script, questions, objections, closing)
- `demo-storyline-auditos.md` — Single-file demo storyline (moved from root; overlaps with modular version)
- `pilot-pack/` — Pilot offer, scope, checklists, client messages
- Commercial assets in `docs/product/auditos-commercial-assets/`
- Execution environment in `docs/product/auditos-first-customer-loop/`
- Live pilot management in `docs/product/auditos-live-pilot-management/`

### For outreach and sales

- Outbound templates in `docs/product/auditos-outbound-kit/`
- Discovery questions in `docs/product/auditos-outbound-kit/discovery-questions.md`
- Follow-up templates in `docs/product/auditos-outbound-kit/follow-up-templates.md`
- Objection handling in `docs/product/auditos-commercial-assets/objection-handling.md`
- Sales pipeline / CRM fields in `docs/product/auditos-sales-ops/`

### For proof and conversion

- Case study capture in `docs/product/auditos-market-proof-system/case-study-publication-workflow.md`
- Proof library in `docs/product/auditos-customer-conversion-reference/customer-proof-library.md`
- Reference request kit in `docs/product/auditos-customer-conversion-reference/reference-request-kit.md`
- Conversion offer in `docs/product/auditos-customer-conversion-reference/paid-conversion-offer-pack.md`
