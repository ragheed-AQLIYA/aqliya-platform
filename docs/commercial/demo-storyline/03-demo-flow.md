## خريطة التدفق

```
[صفحة الديمو] ← ابدأ هنا
    │
    ▼
[رفع TB] ← إبراز: قراءة آلي + التحقق من الاتزان + تقييم الثقة
    │
    ▼
[Mapping] ← إبراز: اقتراحات AI + قبول/تعديل بشري + إعادة بناء القوائم
    │
    ▼
[القوائم] ← إبراز: المركز المالي + الدخل + حقوق الملكية + تنبيه "مسودة"
    │
    ▼
[الإيضاحات] ← إبراز: الإيضاحات المولّدة + النواقص المحددة
    │
    ▼
[الأدلة] ← إبراز: قائمة الأدلة المطلوبة + ربطها بالحسابات
    │
    ▼
[الملاحظات] ← إبراز: التوليد الآلي + تصنيف الخطورة + ربطها بالأدلة
    │
    ▼
[المصادقة] ← إبراز: ٥ فحوصات + تسوية الملاحظات + النسب
    │
    ▼
[سجل التدقيق] ← إبراز: كل إجراء مسجّل + منسوب + مؤرّخ
    │
    ▼
[النشر] ← إبراز: حزمة منشورة + مقفلة + ثابتة
```

## قواعد عامة للديمو

1. **لا تشرح كل شيء.** الديمو مو تدريب — هو عرض للقيمة.
2. **سير العمل هو البطل.** لا تقفز بين الشاشات عشوائياً.
3. **دع النظام يتكلم.** لا تقل "النظام ممتاز" — أرهم كيف يشتغل.
4. **تفاعل مع الحضور.** اسأل: "وش رأيكم؟" "هل هذا يشبه اللي تسوّونه اليوم؟"
5. **الوقت صارم.** إذا طال الديمو عن ١٥ دقيقة، خسرت الانتباه.
6. **جهّز مساراً احتياطياً.** إذا تعطلت شاشة، انتقل للشاشة التالية مباشرة.
7. **اختِم بنسخة.** لا تنهي الديمو وتقول "أي أسئلة؟" — اختِم بخطوة قادمة واضحة.

| احذف | أضف |
