# Financial Statement Review Checklist

## Purpose

This document defines the review checklist for draft financial statements generated by AuditOS. Every statement must pass this checklist before it may be included in the publication package.

## General Review Items

| # | Check | Applies To | Verified |
|---|-------|------------|----------|
| 1 | All accounts mapped | All statements | ☐ |
| 2 | All material accounts included | All statements | ☐ |
| 3 | Opening balances match prior year closing | All statements | ☐ |
| 4 | Comparative data correct (if available) | All statements | ☐ |
| 5 | Classification (current/non-current) correct | SFP | ☐ |
| 6 | Statement balances (A = L + E) | SFP | ☐ |
| 7 | Revenue correctly classified | P&L | ☐ |
| 8 | Expenses correctly classified by function/nature | P&L | ☐ |
| 9 | Gross profit calculation correct | P&L | ☐ |
| 10 | Finance costs match loan balances | P&L | ☐ |

## Review Outcome

| Result | Action |
|--------|--------|
| All checks passed | Statement ready for inclusion in publication package |
| Minor issues found | Correct issues, re-verify checklist |
| Major issues found | Return to drafting, address issues, re-run checklist |
