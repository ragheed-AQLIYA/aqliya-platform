# Legacy Numbered Documentation (Pre-v1.1)

**Status:** Archived — historical reference only

These folders were the original product documentation for AuditOS, written before the AQLIYA v1.1 official reference system was established. They contain domain knowledge about AuditOS workflow, accounting methodology, financial statements, evidence management, and AI governance.

**Why archived:**
- Superseded by the `docs/official/` v1.1 hierarchy as highest authority
- Product positioning in these docs often references AuditOS as if it were the entire AQLIYA platform, which conflicts with v1.1 identity
- The content is still historically valuable but should not override v1.1 references

**Authority:** `docs/official/` remains the highest authority.
**Current reference:** Consult `docs/source-of-truth/` and `docs/systems/auditos/` for up-to-date product documentation.

## Archived Folders

| Folder | Content |
|--------|---------|
| `01-product-foundation/` | AuditOS overview, problem statement, users, workflow, boundaries, disclaimer |
| `04-financial-statements/` | Statement of financial position, profit/loss, equity, cash flows, generation rules |
| `06-evidence-and-review/` | Evidence requirements, review queue, findings, approval, publication |
| `07-ai-governance/` | AI role, human review, confidence, explainability, audit trail, disclaimer |

> Note: Some internal and external documents still reference these folders by their original paths. Those links may be broken. The content remains available here for historical reference.
