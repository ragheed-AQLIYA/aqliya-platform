# Approval Checklist

## Purpose

This document defines the checklist that determines whether an engagement is ready for final approval and publication.

## Pre-Approval Checks

### Data Completeness

| # | Check | Status |
|---|-------|--------|
| 1 | Trial balance uploaded and validated | ☐ |
| 2 | All accounts mapped (no unmapped accounts) | ☐ |
| 3 | Trial balance balances (A = L + E) | ☐ |
| 4 | Trial balance review completed (flags resolved or documented) | ☐ |

### Financial Statements

| # | Check | Status |
|---|-------|--------|
| 5 | Statement of Financial Position generated | ☐ |
| 6 | Statement of Profit or Loss generated | ☐ |
| 7 | Statement of Changes in Equity generated (if data available) | ☐ |
| 8 | Statement of Cash Flows generated (if data available) | ☐ |
| 9 | All statements balance and cross-reference | ☐ |
| 10 | FS review checklist completed (see 04-06) | ☐ |

### Notes

| # | Check | Status |
|---|-------|--------|
| 11 | All required notes generated | ☐ |
| 12 | Accounting policies reviewed and confirmed | ☐ |
| 13 | Note amounts agree with statement amounts | ☐ |
| 14 | Missing data items addressed or documented | ☐ |
| 15 | Notes review checklist completed (see 05-06) | ☐ |

### Evidence

| # | Check | Status |
|---|-------|--------|
| 16 | All required evidence identified | ☐ |
| 17 | Material accounts have evidence uploaded | ☐ |
| 18 | Evidence verified by reviewer | ☐ |
| 19 | Evidence coverage report reviewed | ☐ |
| 20 | Outstanding evidence requests documented | ☐ |

### Findings and Recommendations

| # | Check | Status |
|---|-------|--------|
| 21 | All critical findings resolved or explicitly deferred with partner approval | ☐ |
| 22 | All high findings addressed | ☐ |
| 23 | Recommendations approved by appropriate authority | ☐ |
| 24 | Unresolved findings documented with rationale | ☐ |

### Governance

| # | Check | Status |
|---|-------|--------|
| 25 | All actions attributable to specific users | ☐ |
| 26 | Audit trail complete and reviewable | ☐ |
| 27 | Approvals obtained at required authority levels | ☐ |
| 28 | Governance rule changes documented (if any) | ☐ |
| 29 | All overrides logged with rationale | ☐ |

## Approval Decision

| Result | Condition | Action |
|--------|-----------|--------|
| Approved | All mandatory checks pass | Proceed to publication package |
| Conditionally Approved | Minor non-blocking items remain | Address before or during publication |
| Not Approved | Blocking item exists | Resolve items, re-run checklist |

## Approval Record

```txt
Approval Record
  Engagement:   ABC Company — FY 2026
  Approver:     Partner Name
  Decision:     Approved
  Date:         2026-05-08
  Conditions:   Evidence for Cash balance to be provided within 2 weeks
  Checklist:    29/29 checks passed
```
