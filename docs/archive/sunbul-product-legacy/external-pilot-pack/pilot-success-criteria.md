# Sunbul — Pilot Success Criteria

**Version:** 1.0
**Date:** 2026-05-18
**Classification:** Confidential — Pilot Client

---

## Scoring

Each criterion is scored 0–5:

| Score | Meaning |
|---|---|
| 0 | Not achieved / not tested |
| 1 | Tried but failed |
| 2 | Partially works, major issues |
| 3 | Works but has minor issues |
| 4 | Works well, minor confusion |
| 5 | Works perfectly, no issues |

## Criteria

### 1. Workflow Completion

**Target:** 3+ cases completed Draft → Approved (or Archived)

| Measure | Weight | Score |
|---|---|---|
| Cases created | 30% | ☐ /5 |
| Cases submitted for review | 30% | ☐ /5 |
| Cases approved | 40% | ☐ /5 |

### 2. User Understanding

**Target:** Operators and reviewers can use the system without daily assistance

| Measure | Weight | Score |
|---|---|---|
| Operator created case without help | 30% | ☐ /5 |
| Operator uploaded document without help | 30% | ☐ /5 |
| Reviewer approved case without help | 40% | ☐ /5 |

### 3. Review Traceability

**Target:** Reviewer trusts the audit trail and can trace who did what

| Measure | Weight | Score |
|---|---|---|
| Reviewer can see all actions on a case | 40% | ☐ /5 |
| Reviewer can see return notes | 30% | ☐ /5 |
| Reviewer trusts that data is accurate | 30% | ☐ /5 |

### 4. Document Handling

**Target:** Documents upload, display, and download correctly

| Measure | Weight | Score |
|---|---|---|
| Documents upload successfully | 30% | ☐ /5 |
| Document metadata (name, type, size) is correct | 20% | ☐ /5 |
| Documents download correctly | 30% | ☐ /5 |
| File types are respected (PDF, XLSX, etc.) | 20% | ☐ /5 |

### 5. Export Usefulness

**Target:** Exported PDF is usable for the client's purpose

| Measure | Weight | Score |
|---|---|---|
| PDF generates without errors | 30% | ☐ /5 |
| PDF contains all required sections | 30% | ☐ /5 |
| PDF is useful for the client's workflow | 40% | ☐ /5 |

### 6. Permission Confidence

**Target:** Operators cannot review/approve, reviewers cannot create cases

| Measure | Weight | Score |
|---|---|---|
| Operator cannot approve a case | 40% | ☐ /5 |
| Reviewer cannot create a case | 40% | ☐ /5 |
| Cross-client access is blocked | 20% | ☐ /5 |

### 7. Time Saved

**Target:** Client perceives the workflow as faster than their current process

| Measure | Weight | Score |
|---|---|---|
| Case creation is fast enough | 30% | ☐ /5 |
| Review loop is faster than email | 40% | ☐ /5 |
| Export saves time vs. manual report | 30% | ☐ /5 |

### 8. Willingness to Continue

**Target:** Client wants to continue using Sunbul after the pilot

| Measure | Weight | Score |
|---|---|---|
| Client Operator recommends continuing | 30% | ☐ /5 |
| Client Reviewer recommends continuing | 30% | ☐ /5 |
| Client Decision Maker approves continuation | 40% | ☐ /5 |
