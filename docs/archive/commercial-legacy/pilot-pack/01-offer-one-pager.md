# AuditOS — عرض البايلوت التجريبي

## المنتج
**AuditOS — Financial Intelligence**
نظام تشغيلي محوكم (Governed Workspace) يساعد مكاتب التدقيق والمحاسبة على تحويل ميزان المراجعة إلى:
مسودة قوائم مالية ← إيضاحات ← طلبات أدلة ← ملاحظات مراجعة ← توصيات ← مراجعة ← اعتماد ← تصدير ← نشر الحزمة.

## المبدأ
**الذكاء الاصطناعي يساعد. البشر يقررون. الدليل يحكم.**
كل مخرج من النظام هو مسودة تحتاج مراجعة واعتماد بشري قبل النشر.

## ماذا يشمل عرض البايلوت

| البند | التفاصيل |
|-------|----------|
| المدة | ٣ أسابيع عمل |
| النطاق | مهمة تدقيق واحدة (Engagement) حقيقية ببيانات فعلية |
| المستخدمون | حتى ٥ مستخدمين (مدقق، مراجع أول، شريك) |
| الدعم | تدريب مباشر وجلستان أسبوعياً للمتابعة |
| السعر | مجاني بالكامل — بدون التزام مالي |
| البيئة | بيئة منفصلة ومؤمّنة على خوادم AQLIYA |

## المخرجات المتوقعة من البايلوت
- دورة عمل كاملة: من ميزان المراجعة حتى حزمة النشر النهائية
- مسودة قوائم مالية (المركز المالي، الدخل، التغيرات في حقوق الملكية)
- مسودة إيضاحات مع تحديد النواقص
- قائمة طلبات الأدلة الداعمة مبوّبة حسب الحساب
- ملاحظات المراجعة مع تصنيف الخطورة
- سجل تدقيق كامل (Full Audit Trail)
- تقرير نهاية البايلوت مع توصية الانتقال

## خارج نطاق البايلوت
- تصدير PDF/XLSX (التصدير حالياً بنسق JSON)
- الدخول الموحّد (SSO)
- بوابة العميل الخارجية
- الاستخدام المتزامن لأكثر من مكتب

## العائد المتوقع بعد الانتقال الكامل

| المؤشر | التحسين المتوقع |
|--------|-----------------|
| وقت إعداد القوائم | تخفيض ٤٠–٦٠٪ |
| زمن دورة المراجعة | تخفيض ٣٠–٤٥٪ |
| تغطية الأدلة | تحسين التتبع والاكتمال |
| جودة المراجعة | توحيد المنهجية وتقليل التفاوت |
| أثر التدقيق | تتبع كامل لكل إجراء من المصدر حتى الاعتماد |
| الذاكرة المؤسسية | المعرفة تبقى في النظام لا في الأفراد |

## التالي

- تحديد فريق البايلوت من طرف العميل
