## فريق العميل

### قائد المشروع (Project Lead)
| مسؤولية | تفاصيل |
|----------|--------|
| التنسيق الداخلي | تنسيق عمل الفريق وتوزيع المهام |
| التسليم | ضمان تسليم الملفات المطلوبة في مواعيدها |
| نقطة الاتصال | التواصل مع مدير البايلوت في AQLIYA |
| التغذية الراجعة | تجميع ملاحظات الفريق ورفعها بشكل منتظم |
| القرارات | اتخاذ القرارات المتعلقة بسير البايلوت من طرف العميل |
| الحضور | حضور جميع الجلسات الأسبوعية والتقييم الختامي |

### المدقق / المُعِدّ (Operator)
| مسؤولية | تفاصيل |
|----------|--------|
| رفع البيانات | رفع ميزان المراجعة والتأكد من قراءته |
| توجيه الحسابات | مراجعة اقتراحات الذكاء الاصطناعي وقبولها أو تعديلها |
| رفع الأدلة | تجهيز ملفات الأدلة ورفعها وربطها بالحسابات |
| صياغة الملاحظات | مراجعة الملاحظات الأولية وإضافة ملاحظات إضافية |
| الاستخدام اليومي | المستخدم الأساسي للنظام (٣-٤ ساعات يومياً) |

### المراجع الأول (Reviewer)
| مسؤولية | تفاصيل |
|----------|--------|
| مراجعة التوجيه | التدقيق في توجيه الحسابات وجودة المخرجات |
| مراجعة القوائم | مراجعة مسودة القوائم المالية وجودتها |
| مراجعة الإيضاحات | استعراض الإيضاحات وتحديد النواقص |
| مراجعة الملاحظات | تقييم ملاحظات المراجعة وتصنيف خطورتها |
| تسوية المصادقة | معالجة الملاحظات الناتجة عن فحوصات النظام |
| القرارات المهنية | إصدار الأحكام المهنية على المخرجات |

### الشريك (Partner)
| مسؤولية | تفاصيل |
|----------|--------|
| الاعتماد النهائي | اعتماد مخرجات المهمة أو رفضها |
| النشر | إصدار حزمة التعامل النهائية |
| الجودة | التحقق من جودة المخرجات قبل النشر |
| القرار الاستراتيجي | المشاركة في قرار Go/No-Go |

## قواعد التواصل

1. **كل تواصل يخص البايلوت يمر عبر قائدي المشروع** (عميل + AQLIYA) لضمان عدم تشتت المعلومات.
2. **الاستفسارات اليومية** عبر WhatsApp خلال ساعات العمل (٩ ص - ٥ م).
3. **المشكلات التقنية العاجلة** تبلغ فوراً عبر WhatsApp مع إشعار مباشر.
4. **الملاحظات الجوهرية** توثق بالبريد الإلكتروني لضمان وجود مرجع مكتوب.
5. **الجلسات الأسبوعية** تحضر من جميع أفراد الفريق.
6. **القرارات الملزمة** تصدر من الشريك (عميل) أو مدير البايلوت (AQLIYA) وتوثق كتابياً.

> **نجاح البايلوت يعتمد على وضوح الأدوار والتزام كل طرف بمسؤولياته.**
