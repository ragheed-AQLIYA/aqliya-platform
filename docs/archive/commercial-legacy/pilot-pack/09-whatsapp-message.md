# AuditOS — رسالة واتساب للتواصل الأولي

---

## الإصدار ١: تواصل بارد (Cold Outreach)

```
السلام عليكم أستاذ {الاسم}،

معك {اسمك} من AQLIYA. نعمل على نظام تشغيلي لمساعدة مكاتب التدقيق في تحويل ميزان المراجعة إلى قوائم مالية وملاحظات مراجعة وأدلة داعمة، بسير عمل محوكم ومسجل بالكامل.

أطلقنا برنامج بايلوت تجريبي مجاني (٣ أسابيع) لمكاتب التدقيق، وفكرت إنه قد يكون مفيداً لـ {اسم المكتب}.

النظام لا يستبدل المدقق — يساعده على:
- تسريع إعداد القوائم والإيضاحات
- تتبع الأدلة والملاحظات
- توثيق كل إجراء بسجل تدقيق كامل

هل يناسبك مكالمة ١٥ دقيقة أعرض لك الفكرة؟

مع الشكر،
{اسمك}
AQLIYA
```

## الإصدار ٢: متابعة بعد اتصال أو لقاء (Warm Follow-up)

```
السلام عليكم أستاذ {الاسم}،

سعدت بالحديث معك اليوم عن AuditOS.

كما اتفقنا، هذه مذكرة سريعة:
- بايلوت تجريبي مجاني (٣ أسابيع)
- تدريب الفريق مشمول
- مهمة حقيقية واحدة بنظام محوكم بالكامل
- لا التزام مالي حتى تقتنع بالتجربة

أرسل لك على البريد:
- عرض البايلوت (صفحة واحدة)
- نطاق البايلوت التفصيلي
- قائمة الملفات المطلوبة

إذا تحب نحدد موعد الجلسة التعريفية الأسبوع القادم؟

مع الشكر،
{اسمك}
```

## الإصدار ٣: تأكيد بعد الاتفاق (Confirmation)

```
السلام عليكم أستاذ {الاسم}،

تم استلام الاتفاقيات الموقعة — شكراً لك.

تأكيد الجدول:
- التدريب: {التاريخ} الساعة {الوقت}
- انطلاق البايلوت: {التاريخ}
- المدة: ٣ أسابيع

سأرسل دعوات الحسابات خلال ٢٤ ساعة. لو احتجت أي شيء قبل البداية، أنا موجود.

مع الشكر،
{اسمك}
```

## الإصدار ٤: أثناء البايلوت — تذكير الجلسة الأسبوعية

```
السلام عليكم أستاذ {الاسم}،

تذكير: جلستنا الأسبوعية لمراجعة تقدم البايلوت غداً {اليوم} الساعة {الوقت}.

الرجاء تجهيز:
- أبرز الملاحظات من الأسبوع
- أي صعوبات واجهها الفريق

مع الشكر،
{اسمك}
```

---

## الإصدار ٥: نهاية البايلوت — شكر ودعوة للتقييم

```
السلام عليكم أستاذ {الاسم}،

اكتملت فترة البايلوت اليوم — شكراً لوقتك وفريقك على الالتزام والتجربة.

سأجهز تقرير النتائج خلال يومين، ونحدد اجتماع لتقييم التجربة واتخاذ قرار الاستمرار من عدمه.

هل يناسبك يوم {اليوم} الساعة {الوقت}؟

مع الشكر،
{اسمك}
```

## ملاحظات على الأسلوب
- مهني مباشر، بدون مبالغة
- لا نصوص طويلة — واتساب وسيلة سريعة
- كل رسالة لها هدف واحد واضح
- الرسائل الباردة لا تبيع — تطلب مكالمة فقط
- لا نستخدم وعود غير قابلة للتحقق
