# AuditOS — قائمة الملفات المطلوبة للبايلوت

## قبل بدء البايلوت

يجب على العميل تجهيز الملفات التالية وتسليمها قبل موعد البدء بـ ٣ أيام عمل على الأقل.

---

## ١. ملف ميزان المراجعة (إلزامي)

| المتطلب | التفاصيل |
|---------|----------|
| الصيغة | CSV أو XLSX |
| الأعمدة المطلوبة | `accountCode`, `accountName`, `debit`, `credit` |
| الأعمدة الاختيارية | `openingDebit`, `openingCredit`, `comparativeDebit`, `comparativeCredit` |
| الفترة | سنة مالية واحدة كاملة |
| اللغة | عربي أو إنجليزي |
| الاكتمال | يجب أن يشمل جميع حسابات الأستاذ العام |
| التوازن | مجموع الأرصدة المدينة = مجموع الأرصدة الدائنة |

**تعليمات التصدير من الأنظمة الشائعة:**
- Zoho Books: `Reports → Trial Balance → Export as CSV`
- QuickBooks: `Reports → Trial Balance → Export to Excel`
- SAP: `F.01 → Export → Spreadsheet`
- Microsoft Dynamics: `General Ledger → Trial Balance → Export to Excel`
- أي نظام ERP: صدر تقرير ميزان المراجعة بصيغة CSV أو Excel

---

## ٢. دليل الحسابات (مستحسن)

| المتطلب | التفاصيل |
|---------|----------|
| الصيغة | PDF أو XLSX |
| المحتوى | دليل حسابات الشركة كاملاً مع المستويات (حساب رئيسي، فرعي، تحليلي) |
| ملاحظة | يساعد في تحسين دقة توجيه الحسابات (Mapping) |

---

## ٣. القوائم المالية للعام السابق (مستحسن)

| المتطلب | التفاصيل |
|---------|----------|
| الصيغة | PDF أو XLSX |
| المحتوى | القوائم المالية للسنة السابقة (إن وجدت) |
| الفائدة | يستخدمها النظام للمقارنة واكتشاف التغيرات الجوهرية |

---

## ٤. سياسات محاسبية (اختياري)

| المتطلب | التفاصيل |
|---------|----------|
| الصيغة | PDF أو Word |
| المحتوى | ملخص السياسات المحاسبية المتبعة في الشركة |
| الفائدة | تساعد الذكاء الاصطناعي في صياغة إيضاحات أدق |

---

## ٥. ملفات الأدلة الداعمة (أثناء البايلوت)

خلال مرحلة الأدلة، سيطلب النظام أدلة داعمة حسب كل منطقة حسابية. يُفضّل تجهيز الملفات التالية مسبقاً:

- [ ] عقود الإيجار
- [ ] عقود القروض والتسهيلات
- [ ] كشوف حسابات بنكية
- [ ] محاضر مجلس الإدارة
- [ ] عقود العملاء الرئيسيين
- [ ] عقود الموردين الرئيسيين
- [ ] سجل الأصول الثابتة
- [ ] الجرد الفعلي نهاية العام
- [ ] التسويات البنكية
- [ ] كشوف المديونيات المدينة والدائنة المفصّلة

## ملاحظات هامة
- جميع الملفات تعامل بسرية تامة وفق اتفاقية NDA الموقعة
- البيانات لا تشارك مع أي طرف ثالث ولا تستخدم لتدريب نماذج الذكاء الاصطناعي
- يتم حذف بيانات البايلوت من البيئة بعد ٣٠ يوماً من انتهاء البايلوت ما لم يوقع العميل عقد الاشتراك

---

> **تسليم هذه الملفات قبل البدء بـ ٣ أيام عمل شرط لانطلاق البايلوت في الموعد المحدد.**
