## Trust Language

- **"AI assists. Humans decide. Evidence governs."** — Core trust principle
- **"Human-reviewed AI"** — Not "AI-powered" or "AI-driven"
- **"Trust-driven systems"** — Not "trustless" or "zero-trust"
- **"Enterprise Systems Builder"** — Not "AI company" or "SaaS provider"

## Forbidden Copy

- "Fully automated audit" — AuditOS requires human review
- "Replaces auditors" — AuditOS assists, does not replace
- "Production-ready" — Unless explicitly validated
- "Commercial-ready" — Unless explicitly validated
- "Revolutionary" — Overclaimed
- "Game-changing" — Overclaimed
- "Disruptive" — Overclaimed
- "Next-gen" — Generic and overused
- "AI-powered" — Prefer "human-reviewed AI" or "intelligence-assisted"

## Arabic Copy Rules

- Arabic is the primary language for marketing
- English system names (AuditOS, DecisionOS, SalesOS) are preserved in Arabic contexts
- Technical terms can remain in English (ICP, scoring, pipeline, etc.)
- Dates use Hijri/Gregorian format based on context
- Numbers use Arabic-Indic (٠-٩) or Western (0-9) based on data type
