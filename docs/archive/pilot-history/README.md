# Archived Pilot History

**Status:** Archived — historical record only

These are individual pilot session reports, run records, and historical pilot documentation that document the AQLIYA AuditOS pilot execution.

**Why archived:**
- Individual session reports are historical records, not active operational docs
- Active pilot runbooks, checklists, and customer-facing documents remain in `docs/pilot/`
- The content is retained for audit trail and historical reference

**Authority:** Active pilot operations are documented in `docs/pilot/`.
