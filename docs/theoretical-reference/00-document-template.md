# AQLIYA Theory Document Template

Every theoretical document in the AQLIYA Reference System must follow this template.

## Frontmatter

```yaml
---
title:        # Full document title
document_id:  # e.g., 01.01
status:       # Not Started | Draft | Reviewed | Approved | Active Reference | Needs Revision | Deprecated
owner:        # Person responsible for this document
version:      # Semantic version, e.g., 0.1
last_updated: # Date of last update
priority:     # Critical | High | Medium | Low
depth_level:  # Level 1 — Core Doctrine | Level 2 — Domain Theory | Level 3 — Model/Framework | Level 4 — Definition | Level 5 — Narrative
related_documents: # Comma-separated list of related document IDs
---
```

## Template Structure

```
# Title

## 1. Purpose
Why this document exists and what it intends to achieve.

## 2. Thesis
The core argument or position this document takes.

## 3. Problem
The specific problem that motivates this theory.

## 4. Why Existing Systems Fail
Analysis of how current approaches are inadequate.

## 5. AQLIYA Philosophy
How AQLIYA's approach differs from existing solutions.

## 6. Core Principles
The fundamental principles that guide this area.

## 7. Key Concepts
The essential concepts and terminology introduced.

## 8. Operational Implications
How this theory affects daily operations.

## 9. Product Implications
How this theory should influence product decisions.

## 10. Architecture Implications
How this theory affects system architecture.

## 11. Governance Implications
How this theory affects governance and compliance.

## 12. AI / Intelligence Implications
How this theory affects AI and intelligence design.

## 13. UX Implications
How this theory affects user experience.

## 14. Commercial Implications
How this theory affects commercial strategy and pricing.

## 15. Anti-Patterns
Common mistakes and wrong approaches related to this topic.

## 16. Examples
Real or illustrative examples of this theory in practice.

## 17. Enterprise Impact
The measurable impact on enterprise customers.

## 18. Long-Term Strategic Importance
Why this matters for AQLIYA's long-term positioning.

## 19. Related Documents
Links to related documents by ID.

## 20. Version History
| Version | Date | Author | Changes |
|---------|------|--------|---------|
| 0.1     |      |        | Initial draft |
```
