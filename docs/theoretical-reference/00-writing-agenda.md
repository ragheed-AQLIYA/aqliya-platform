## Dependency Notes

- Foundation docs (Phase 1) block all subsequent phases
- AuditOS theory (Phase 3) depends on Financial Intelligence Thesis (04.01) and AuditOS Thesis (05.01)
- Governance docs (Phase 4) depend on Governance & Trust Thesis (08.01)
- Narratives (Phase 7) depend on Foundation docs and AuditOS theory
- Long-term theory (Phase 8) should not be written before Phase 1-3 are stable

## Cadence

- Weekly: Write 1-2 core/domain documents
- Biweekly: Review dependencies and consistency
- Monthly: Review strategic alignment and update agenda
- Quarterly: Review entire theoretical system
