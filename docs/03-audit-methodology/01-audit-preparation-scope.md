# Audit Preparation Scope

## Purpose

This document defines the scope of audit preparation activities that AuditOS supports. It clarifies what the system does and does not cover within the audit engagement lifecycle.

## AuditOS Role in the Audit Lifecycle

AuditOS supports the preparation and review phases of an audit engagement — from data intake through to evidence review and finding preparation. It does not replace the audit execution or opinion phases.

```txt
Full Audit Lifecycle:

1. Engagement Acceptance         ← Outside AuditOS scope (firm process)
2. Risk Assessment                ← Outside AuditOS scope (professional judgment)
3. Audit Planning                 ← Outside AuditOS scope (professional judgment)
4. Audit Preparation              ← AuditOS supports
   a. Trial Balance Intake
   b. Account Mapping
   c. Financial Statement Drafting
   d. Notes Drafting
   e. Evidence Requirement Identification
5. Evidence Collection            ← AuditOS supports (tracking, not collection)
6. Audit Execution                ← Outside AuditOS scope (fieldwork)
7. Review and Findings            ← AuditOS supports
8. Conclusion and Opinion         ← Outside AuditOS scope (professional judgment)
9. Reporting                      ← AuditOS supports (draft outputs)
```

## In-Scope Audit Preparation Activities

| Activity | Description |
|----------|-------------|
| Trial balance intake | Import, validate, and structure client financial data |
| Account mapping | Map client COA to standard financial model |
| Financial statement drafting | Generate draft statements from mapped data |
| Notes drafting | Generate draft notes with available data |
| Evidence requirement identification | Determine supporting evidence needed |
| Review finding creation | Identify potential issues from data analysis |
| Finding management | Track findings through review lifecycle |
| Recommendation drafting | Generate draft recommendations with evidence links |
| Approval workflow | Route findings through governed review and approval |
| Publication package | Generate client-facing output from approved items |
| Audit trail | Maintain complete record of all actions |

## Out-of-Scope Audit Activities

| Activity | Reason for Exclusion |
|----------|---------------------|
| Engagement acceptance and continuance | Firm-specific professional decision |
| Risk assessment and materiality determination | Professional judgment |
| Audit planning and program design | Professional judgment |
| Fieldwork and substantive procedures | Professional execution |
| Sample selection | Professional methodology |
| Audit opinion formulation | Professional conclusion — cannot be automated |
| Final report signing | Professional responsibility — cannot be delegated |
| Regulatory filing | Entity-specific regulatory process |

## AuditOS Outputs vs. Audit Deliverables

| AuditOS Output | Related Audit Deliverable | Status |
|----------------|--------------------------|--------|
| Draft financial statements | Final financial statements | Draft — requires management and auditor approval |
| Draft notes | Final notes to financial statements | Draft — requires review and supplementation |
| Evidence requirements | Audit evidence | Tracking tool — collection is separate |
| Review findings | Audit findings and adjustments | Draft — requires professional assessment |
| Recommendations | Recommended adjustments | Draft — requires management agreement |
| Traceability report | Audit documentation | Supporting — not a replacement for audit file |
