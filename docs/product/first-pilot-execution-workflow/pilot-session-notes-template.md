# Pilot Session Notes Template — AQLIYA

## Purpose

Document every customer-facing session during the pilot. Creates an auditable record of what was shown, what was said, what feedback was received, and what decisions were made.

## When to Use

- After every demo, review, or working session with the customer
- Within 24 hours of the session

## Inputs

- Pilot command center (`pilot-command-center.md`)
- Demo and pilot claims script (`pilot-control-pack/demo-and-pilot-claims-script.md`)
