## Common Patterns to Watch For

| Pattern | What to Listen For |
|---|---|
| **Label confusion** | "What does this button mean?" "I don't know what to click next" |
| **Navigation confusion** | "How do I go back to the list?" "Where did my case go?" |
| **Permission confusion** | "Why can't I see that button?" "I thought I could approve" |
| **Status confusion** | "Is the case approved now?" "What does this status mean?" |
| **File confusion** | "Where is my document?" "How do I download it?" |
| **Export confusion** | "Where is the PDF?" "What does this report contain?" |
| **Trust concern** | "Is my data safe?" "Who else can see this?" |
| **Missing feature** | "I wish it could..." "Can it do..." |
