## Purpose

هذا السجل يلتقط التعلم الحقيقي من أول العملاء، وليس فقط status التشغيلية.

## Weekly Learning Prompts

1. ما الذي فهمناه عن buyer الحقيقي هذا الأسبوع؟
2. ما use case التي ظهر أنها الأقوى؟
3. ما message التي resonated أكثر؟
4. ما الاعتراض الذي تكرر؟
5. ما الذي يجب تحسينه في demo / onboarding / pilot scope؟

## Learning Categories

1. ICP learning
2. messaging learning
3. workflow learning
4. objection learning
5. paid conversion learning

## Rule

إذا لم نخرج من كل أسبوع بتعلم واحد واضح على الأقل، فنحن نجمع activity أكثر مما نجمع signal.
