# AQLIYA AuditOS Deal Stage Definitions

## Discovery

تم بدء نقاش فعلي لفهم workflow والألم والfit.

## Demo Scheduled

تم الاتفاق على demo قادمة مع audience واضحة.

## Demo Completed

تم تنفيذ demo ويوجد next-step discussion.

## Pilot Proposed

تم اقتراح pilot أو proof-of-value مع scope أولية.

## Pilot Active

الـ pilot بدأت فعليًا أو تم الاتفاق التشغيلي عليها.

## Proposal / Commercial Review

الفرصة دخلت مرحلة proposal أو commercial discussion.

## Negotiation

هناك تفاوض فعلي على الشكل التجاري أو النطاق أو الخطوة التالية.

## Won

تم الاتفاق التجاري أو اعتماد الخطوة الرسمية التالية.

## Lost

توقفت الفرصة أو اختار العميل عدم الاستمرار.
