# AQLIYA AuditOS Pilot Closeout Checklist

Source of truth:

1. `docs/product/auditos-sales-ops/pilot-scoring-sheet.md`
2. `docs/product/auditos-live-pilot-management/pilot-success-review.md`

## Purpose

هذا checklist لإغلاق أي pilot بشكل منظم قبل اتخاذ القرار التالي.

## Closeout Checks

1. هل تم تقييم success criteria؟
2. هل تم جمع evidence كافية؟
3. هل تم تسجيل stakeholder feedback؟
4. هل تم تحديث issue/risk register؟
5. هل تم إعداد post-pilot decision memo؟
6. هل next commercial decision واضحة؟

## Go / No-Go Outcome

### Go

إذا:

1. fit confirmed
2. buyer engaged
3. value evidence موجودة
4. next step agreed

### Conditional Go

إذا:

1. value ظهرت جزئيًا
2. scope يحتاج tightening
3. extension صغيرة قد تكون مفيدة

### No-Go

إذا:

1. use case غير مناسبة
2. sponsor غير ملتزم
3. value غير واضحة
4. risk / blocker أعلى من الفائدة المتوقعة

## Final Closeout Note

في نهاية الإغلاق، سجّل:

1. final decision
2. owner
3. date
4. next step
5. lesson learned
