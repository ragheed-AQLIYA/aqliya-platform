# AuditOS Light CRM Import Format

Source of truth:

1. `docs/product/auditos-sales-ops/crm-qualification-fields.md`
2. `docs/product/auditos-live-pilot-management/pilot-master-tracker.md`

## Purpose

صيغة خفيفة يمكن استخدامها في Notion / Airtable / HubSpot / Pipedrive لتتبع pilot customers.

## Suggested Columns

1. Account Name
2. Contact Name
3. Contact Role
4. Segment
5. Internal Owner
6. Pilot ID
7. Pilot Status
8. Start Date
9. Planned End Date
10. Use Case
11. Main Pain
12. Success Criteria
13. Risk Level
14. Next Action
15. Next Action Owner
16. Next Action Date
17. Post-Pilot Decision
18. Notes

## Usage Rule

استخدم هذه الصيغة إذا أردت:

1. تتبع الحسابات في sheet بسيطة
2. رفعها إلى CRM خفيفة
3. الاحتفاظ بـ one-row-per-pilot structure

## Minimum Required Columns

إذا أردت نسخة أبسط جدًا، استخدم فقط:

1. Account Name
2. Segment
3. Internal Owner
4. Pilot Status
5. Main Pain
6. Next Action
7. Next Action Date
8. Post-Pilot Decision
