## Credibility Milestones

### Milestone 1: First Evidence
- [ ] أول pilot مكتملة
- [ ] أول 5 proof assets في الـ index (أي level)
- [ ] أول learning موثقة في first-revenue-learning-memo

### Milestone 2: First Paid Proof
- [ ] أول paid conversion مكتمل
- [ ] أول proof assets بمستوى `PU` أو `AN`
- [ ] أول عميل مؤهل كـ reference (أي tier)

### Milestone 3: First Published Proof
- [ ] أول case study منشورة (Tier 1 أو 2)
- [ ] 10+ proof assets في الـ index
- [ ] عميلان على الأقل في reference program

### Milestone 4: Market Pattern
- [ ] 3 paid conversions مكتملة
- [ ] pattern واضح في الـ segment أو use case الأنجح
- [ ] conversion time يتحسن
- [ ] materials sales قابلة للاستخدام بثقة

### Milestone 5: Investment-Ready Proof
- [ ] case study منشورة + reference accounts نشطة
- [ ] 20+ proof assets
- [ ] قصة نمو واضحة من أول 5 عملاء
- [ ] market signals موثقة وقابلة للعرض على مستثمرين

## Weekly Dashboard Update Checklist

- [ ] هل أضيف أي proof asset جديد هذا الأسبوع؟
- [ ] هل تغير usage level لأي أصل موجود؟
- [ ] هل تغيرت حالة أي عميل في reference program؟
- [ ] هل اكتمل أي conversion هذا الأسبوع؟
- [ ] هل ظهرت أي objections جديدة أو signals مهمة؟

## Rules

1. هذا الـ dashboard لا يُشارك خارجيًا — هو أداة داخلية للـ founder و commercial lead
2. لا تملأ المؤشرات بتقديرات — استخدم فقط ما هو موثق فعليًا
3. الـ dashboard صادق — إذا كانت الأرقام منخفضة، هذا طبيعي في البداية
4. الهدف ليس أرقام كبيرة بسرعة — بل نمط نمو حقيقي وموثوق
5. كل milestone له قيمة فقط إذا كان مبنيًا على evidence حقيقية
