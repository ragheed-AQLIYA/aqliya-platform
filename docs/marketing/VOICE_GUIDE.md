## نبرة حسب الدور

| الدور | النبرة | مثال hook |
|-------|--------|-----------|
| قيادة | مختصر، قرار استثمار | «شوفوا الملخص والديمو — ثم قرروا إذا يستحق وقت فريقكم.» |
| CFO | أرقام، مصدر، اعتماد | «من ميزان المراجعة إلى ملف جاهز — وكل رقم له ورقة مصدر.» |
| مقاولات | امتثال، مواعيد | «موردون وإنفاق ومحتوى محلي — في مكان واحد بدل جداول متفرقة.» |
| تقنية | صدق، بدون مبالغة | «نشر سحابي جاهز. خاص أو معزول — نناقشه إذا احتجتموه.» |
| تدقيق | سمعة، اعتماد | «ملف ارتباط كامل — لا يُعدَّل بعد توقيع الشريك.» |
| مشتريات | ملفات، معايير | «PDF جاهز: أمن، نطاق، ومقارنة — للجنة الترسية.» |
| حكومة | مساءلة، تقارير | «تقارير امتثال ومحتوى محلي جاهزة للمراجعة التنظيمية.» |
