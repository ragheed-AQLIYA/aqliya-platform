# Notes Review Checklist

## Purpose

This document defines the review checklist for draft notes to financial statements generated by AuditOS.

## General Review Items

| # | Check | Verified |
|---|-------|----------|
| 1 | All required notes for this entity type are present | ☐ |
| 2 | Note numbers and titles match statement references | ☐ |
| 3 | Note amounts agree with financial statement amounts | ☐ |
| 4 | Comparative period data is complete and accurate | ☐ |
| 5 | Accounting policies are appropriate for the entity | ☐ |
| 6 | Disclosure requirements are met for the applicable standard | ☐ |

## Accounting Policies Note

| # | Check | Verified |
|---|-------|----------|
| 7 | Basis of preparation correctly identifies accounting standard | ☐ |
| 8 | Measurement basis stated (historical cost, fair value, etc.) | ☐ |
| 9 | Functional and presentation currency stated | ☐ |
| 10 | All significant accounting policies disclosed | ☐ |
| 11 | Policies match entity's actual accounting practices | ☐ |
| 12 | Changes in accounting policies disclosed (if any) | ☐ |
| 13 | Judgments and estimates disclosed | ☐ |

## Asset Notes

| # | Check | Verified |
|---|-------|----------|
| 14 | Cash and cash equivalents components disclosed | ☐ |
| 15 | Restricted cash identified (if applicable) | ☐ |
| 16 | Receivables net of impairment | ☐ |
| 17 | Impairment methodology and assumptions disclosed | ☐ |
| 18 | Inventory composition disclosed | ☐ |
| 19 | Inventory costing method and NRV policy stated | ☐ |
| 20 | PPE movement schedule complete (cost, additions, disposals, depreciation) | ☐ |
| 21 | Depreciation method and useful lives disclosed | ☐ |
| 22 | Impairment assessment disclosed (if applicable) | ☐ |

## Liability Notes

|---|-------|----------|

| 24 | Loan terms, interest rates, and maturity disclosed | ☐ |

## Specialized Disclosures

| # | Check | Verified |
|---|-------|----------|
| 35 | Related party transactions and balances disclosed | ☐ |
| 36 | Key management compensation disclosed (if applicable) | ☐ |
| 37 | Zakat base and provision disclosed (if applicable) | ☐ |

## Missing Data Items

| # | Check | Verified |
|---|-------|----------|
| 41 | All flagged missing data items have been addressed | ☐ |
| 42 | Items that cannot be provided have been documented | ☐ |
| 43 | Reviewer has confirmed that disclosure is complete | ☐ |
| 44 | Management has reviewed and confirmed note disclosures | ☐ |

## Review Outcome

| Result | Action |
|--------|--------|
| All checks passed | Notes ready for inclusion in publication package |
| Minor issues (typos, formatting) | Correct and proceed |
| Moderate issues (missing data, incomplete notes) | Address before publication |
| Major issues (incorrect policies, material misstatements) | Return to drafting, re-run checklist |
